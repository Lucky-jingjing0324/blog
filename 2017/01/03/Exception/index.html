异常处理简介

Throwable：所有不正常类都继承Throwable

Throwable包括Error和Exception

Erro：虚拟机错误（VituolMachineError）和线程死锁（ThreadDeath）

Exception：编译，环境，用户操作输入出现异常

Exception包括非检索异常（RuntimeException）和检索异常

一，非检索异常包括空指针异常（NullPointerException），数组下标越界异常（ArrayIndexOutOfBoundsException），类型转换异常（ClassCastException），算术异常（ArithmeticException）等等

空指针异常
     String str = Null；
     System.out.println(str.length());

数组下标越界异常
     int[] ary={1,2,3};
    for(int i = 0;i <= 3;i++){
        System.out.println(ary[i]);
    }

类型转换异常
     class Animal{
    }
    class Dog extends Animals{
    }
    class Cat extends Animals{
    }
    Public class Test{
        Public Static void main （String[] args）{
            Animals a1 = new Dog();
            Animals a2 = new Cat();
            Dog d1 = (Dog)a1;
            Dog d2 = (Dog)a2;
    }

算术异常
    int one = 12;
    int two = 0;
    Syetem.out.println(one/two);

二，检查异常包括文件异常（IOException）和SQL异常（SQLException）
