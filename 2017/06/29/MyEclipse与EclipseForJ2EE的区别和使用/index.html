MyEclipse与EclipseForJ2EE的区别和使用

发表于:2017-06-29，更新于:2017-06-30，By jingjing
大纲

一.MyEclipse与Eclipse的使用是一样的，发布时不一样的

Eclipse运行服务时不会拷贝WebContent下的文件D:\apache-tomcat-7.0.47\webapps
workspace\.metadata\.plugins\org.eclipse.wst.server.core\tmp0\wtpwebapps\下多一个项目名里面是webContext里的文件

MyEclipse运行服务时会拷贝WebRoot下的文件到D:\apache-tomcat-7.0.47\webapps

二.MyEclipse
1.MyEclipse

D:\apache-tomcat-7.0.47\webapps文件下文件有：
docs
examples
host-manager
manager
ROOT

2.MyEclipse 启动服务

方法一：项目名右键->Run As ->MyEclipse Server Application

方法二：(1)启动tomcat服务

    "像小冰箱的图标"Tocat 7.x-> Start

    (2)部署
        1. 单击像小冰箱的图标的左边图标
            《"像小冰箱的图标的左边图标叫Deploy MyEclipse J2EE Project to Server...(部署MyEclipse J2EE 项目到服务器)》
        2. Project:选择项目 ->Add 
        3. Server:选择Tomcat 7.x 
        4. Deploy location：D:\apache-tomcat-7.0.47\webapps\项目名自动将WebRoot下文件拷贝到D:\apache-tomcat-7.0.47\webapps下并且把WebRoot的名字改成项名

三.Eclipse 

1.Window ->Preferences ->Server ->Runtime Environments(运行环境)->Add->D:\apache-tomcat-7.0.47

2.Workspace文件下多了一个Servers文件
    Workspace->Servers-> Tomcat v7.0 Server at localhost -> config
    catalina.policy
    catalina.properties
    context.xml
    server.xml
    tomcat-users.xml
    web.xml

四.公司开发部署tomcat上

部署方法一：
1.webContext下的文档压缩包
    项目名右键->Export->WAR file ->Web project:test ->Browse:D:\test.war (war:web application 文档) ->Finish


2.复制test.war文件到D:\apache-tomcat-7.0.47\webapps

3.手动启动tomcat应用
          1.ctrl+R，输入cmd->确定

    2.C:\>d: 进入d盘，因为Tomcat安装在d盘

    3.D:\>cd apache-tomcat-7.0.47

    4.D:\apache-tomcat-7.0.47>cd bin

    5,D:\apache-tomcat-7.0.47\bin>startup.bat  这时出现Tomcat窗口

    自动将test.war文件解压

4.打开浏览器输入http://localhost:8080/项目名字->回车就可以访问到了

部署方法二：
1.手动启动tomcat应用
2.修改用户名和密码
    打开浏览器输入http://localhost  
    单击Manager App(管理应用程序)->确定 出现401页面
    D:\apache-tomcat-7.0.47\conf\tomcat-users.xml右键编辑
    添加在401页面上出现的语句

    <role rolename ="manager-gui"/>
    <user username="tomcat" password="tomcat" roles="manager-gui/>


3.重启服务器
    输入sgutdown.bat ->startup
    打开浏览器输入http://localhost  
    单击Manager App(管理应用程序)->确定
    进入页面找到Deploy（部署）->WAR file to deploy->浏览找到        test.war文件->Deploy 然后就部署上去了->点击test就可以访问了
