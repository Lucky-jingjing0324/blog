myeclipse项目移植到eclispe项目

发表于:2017-06-29，更新于:2017-06-29，By jingjing
大纲

1.打开eclipse项目文件

2.将myeclipse项目文件拷贝到eclipse的workspace

3.eclipse中右键Import->Import…->General->Existing Project into Workspace ->Browse…（选择项目的路径）->Finish

4.导入的项目引用的库是myeclipse
(1)右键-> Build Path ->Congigure Build Path… ->选中melibrary.com.genuitec.eclipse.j2eedt.core.MYECLIPSE_JAVAEE_5_CONTAINER -> Remove
(2)选中EAR Libraries ->Remove

5.导入servletjar包
Add External JARs…(导入外部的jar包)->D:\apache-tomcat-7.0.47\lib\servlet-api.jar

必要的jarbao
servlet-api.jar
el-api.jar
jasper-el.jar
jasper-el.jar
jsp-api.jar
