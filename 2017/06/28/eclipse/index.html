eclipse创建workspace

发表于:2017-06-28，更新于:2017-06-28，By jingjing
大纲

1.建立一个文件夹workspace

2.启动eclipse

3.file -> Switch workspace ->other…->E:\workspace->ok

4.Window ->Prefences->Server ->Runtime Environments ->Add…->
Apache Tomcat v7.0 ->Next ->Browse…->D:\apache-tomcat-7.0.47
->Finish

5.import项目 General->Existing Project into Workspace(导入已有的项目)->Finish

6.项目右键Run AS ->Run on Service ->Apache Tomcat v7.0 ->Finish
